## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. **Navigate to the landing page directory**

   ```bash
   cd landing-page
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

   ```
   http://localhost:3000
   ```

